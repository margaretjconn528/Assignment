## ✨ Features

- **Music Playback**: Play, pause, skip, and seek through a curated playlist.
- **Modern UI**: Premium dark-themed interface with smooth transitions.
- **YouTube Integration**: Integrated mobile-optimized YouTube view within the app.
- **Custom Controls**: Tailored audio controller with duration and progress tracking.

## 🛠️ Technologies

- **Flutter**: Cross-platform framework.
- **Provider**: State management.
- **Audioplayers**: For local and remote audio playback.
- **WebView Flutter**: Integrated web browsing experience.

## 🚀 How to Run

1. **Clone the repository**:
   ```bash
   git clone https://github.com/atikur01/Xylophone-App-Media-Player.git
   ```
2. **Install dependencies**:
   ```bash
   flutter pub get
   ```
3. **Run the app**:
   ```bash
   flutter run
   ```
